# 🏥 Hospital Analytics Dashboard

## 📌 Project Overview
This project presents an end-to-end Hospital Analytics Dashboard built using Excel, SQL, Python, and Power BI. The objective is to analyze hospital operations, patient trends, department performance, and revenue to support data-driven decision-making.

## 🛠️ Tools & Technologies
- Microsoft Excel
- SQL
- Python (Pandas, Matplotlib)
- Power BI

## 📊 Dashboard Features
- Executive Summary
- Total Patients
- Total Revenue
- Average Bill Amount
- Average Length of Stay
- Monthly Revenue Trend
- Department-wise Revenue
- Doctor Performance
- Gender Distribution
- Treatment Revenue
- Patient Trend
- Interactive Slicers

## 📈 Key Insights
- Identified top revenue-generating departments.
- Analyzed doctor workload and patient volume.
- Monitored monthly revenue trends.
- Evaluated treatment performance.
- Supported management decision-making using interactive dashboards.
